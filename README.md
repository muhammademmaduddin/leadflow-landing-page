# LeadFlow — Business Landing Page with Lead Capture Form

LeadFlow is a responsive business landing page designed for agencies, consultants, service providers, and small businesses that want to convert website visitors into potential leads.

## Live Demo

Live Website: https://leadflow-by-emmaduddin.netlify.app

## Project Overview

This project includes a modern landing page layout with a clear hero section, service highlights, business results section, and a lead capture form. The form uses JavaScript validation to check user input before showing a success message.

## Features

* Responsive landing page design
* Modern hero section
* Service cards
* Business results/statistics section
* Lead capture form
* JavaScript form validation
* Success/error message handling
* Clean and organized code structure

## Technologies Used

* HTML
* CSS
* JavaScript
* GitHub
* Netlify

## Project Purpose

The purpose of this project is to show how a simple and professional landing page can help a business present its services clearly and collect client inquiries through a lead form.
